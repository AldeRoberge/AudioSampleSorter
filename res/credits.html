This software is developed as a Java practice to create a simple to use audio file sorter.

It is inspired by FL Studio's auto playing file browser and lack of sorting tools for large audio libraries.


Here are the awesome libraries and tools used to overcome development challenges :


Credits are generated by the awesome online HTML editor html-online.com/editor/

Property files is handled by the awesomely simple Commons-configuration library

jnativehook is used to listen for global key imputs

Sounds are played by the best in class Java audio player, from nonetheless than JavaLayer

And finally StackOverflow to support my lazy coding practices
